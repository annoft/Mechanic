# Mechanic Tools

This directory contains development tools downloaded or generated by `mech setup`.

## Tools

| Tool | Purpose | Version |
|------|---------|---------|
| luacheck.exe | Lua linter | 0.23.0 |
| stylua.exe | Lua formatter | 2.3.1 |
| lua.exe | Lua runtime (optional) | 5.1.5 |
| busted.bat | Busted wrapper generated from LuaRocks paths | 2.2.0-1 recommended |

Generated binaries and wrappers are ignored by git. Keep only this README and
`checksums.json` in source control.

## Setup

```bash
# Download all required tools
mech setup

# Verify installation
mech setup --verify

# Force re-download without overwriting known-good tools unless the new
# download passes checksum verification
mech setup --force

# Regenerate the local Busted wrapper after installing LuaRocks/Busted
mech setup-busted
```

## Manual Installation

If automatic download fails, you can download manually:

- **Luacheck**: https://github.com/mpeterv/luacheck/releases
- **StyLua**: https://github.com/JohnnyMorganz/StyLua/releases
- **Lua 5.1**: use the Lua bundled with your LuaRocks Windows install, or a
  Lua 5.1 runtime matching the architecture used to build Busted dependencies.

Place executables and required DLLs in this directory. For Busted, install it
with LuaRocks and run `mech setup-busted`; do not hand-edit `busted.bat`.
